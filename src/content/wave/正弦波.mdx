### 正弦波

波源が単振動をする場合，媒質の各点は波源よりも遅れて単振動するので，**正弦波**が生じる．各点における，元の位置からのズレは，**変位**という．

### 図について

横波の場合のモデル．この図では，角振動数と波の速さを変更することができ，波長は適切な値が設定されます．スライダを動かすことで，$v=fλ$についてよく分かるでしょう．

### 波の要素

- **山**：波形の最も高いところ
- **谷**：波形の最も低いところ
- **振幅**$A$：山の高さ・谷の深さ
- **波長**$\lambda$：波一つ分の長さ．山と山の間など．
- **波の速さ**$v$：山や谷が進む速さを指す．
- **周期**$T$：媒質の各点が１回振動するのにかかる時間
- **振動数**$f$：１秒あたりに振動する回数

これらの要素の関係式は，以下の通り．

$$
v = f \lambda
$$

$$
f = \frac {1} {T}
$$

### 正弦波の具体例

- **音波**：縦波だが，正弦波で表される．
- **電磁波**：光（可視光），赤外線，電波など．

### 正弦波の式

波源は単振動しており，波は$x$軸の正の向きに速さ$v  \text{[m/s]}$で伝わると仮定する．<br />
正弦波の振幅$A \text{[m]}$，周期$T \text{[s]}$，波長$\lambda  \text{[m]}$とおく．<br />
原点$x=0$にある波源の，時刻$t  \text{[s]}$での変位$y  \text{[m]}$は，次のように表される．<br />
$$
y = A \sin \frac { 2 \pi } { T } t
$$
ここで，位置$x \text{[m]}$（点Pとする）にある媒質の，時刻$t  \text{[s]}$における変位$y  \text{[m]}$を考える．<br />
点Pに原点の振動が伝わるのにかかる時間$t _ { 0 } \text{[s]}$は，次のように表される．<br />
$$
t _ { 0 } = \frac { x } { v } \text{[s]}
$$
したがって，時刻$t \text{[s]}$での点Pの変位$y$は，時刻$(t-t_{0}) \text{[s]}$での原点の変位と同じである．ゆえに，<br />
$$
y = A \sin \frac { 2 \pi } { T } ( t - t _ { 0 } ) = A \sin \frac { 2 \pi } { T } ( t - \frac { x } { v } )
$$
ここで，$v=\frac{\lambda}{T}$より，**正弦波の式**は次のようになる．
$$
y = A \sin 2 \pi ( \frac { t } { T } - \frac { x } { \lambda } )
$$
